**Copila - AI 客製化功能平台**

- **系統內容：** 提供 AI 對話與繪圖，以及由 AI 生成多項客製化圖表等功能。
- **技術架構：** Angular PWA, Laravel Swoole Octane, Ubuntu + Nginx
- **技術摘要：**
    - 以 Laravel Swoole Octane 實踐 WebSocket Sever
    - 搭配 Angular 呈現即時通訊功能，提供AI繪圖並實踐線上手繪功能
    - 圖片在 Nginx 以內網IP＋反向代理方式，向 Laravel 主機取圖後，在Nginx即時運算並回覆多種尺寸與檔案類型，分散系統運算資源
    - 串接 OPENAI API 實現 AI版 心智圖、流程圖、曼陀羅九宮格思考術
    - MySQL 與 Redis 實現自動同步及讀寫分離策略，有效提升資料庫效能與系統彈性。
- **貢獻：** 單人負責前端、後端及設計。
- **成就：** 使用 Lighthouse 檢測，各項目接近滿分，反映系統的優異性能和效率。
