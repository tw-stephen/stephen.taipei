# Stephen Taipei - Personal Image Website

This is a personal image website for Stephen Chuang, a Senior Full Stack Engineer.
It is built with React, Vite, and Tailwind CSS, featuring a bright, professional design as requested.

## Tech Stack

- **Framework:** React + Vite
- **Styling:** Tailwind CSS (Bright/Professional Theme)
- **Icons:** Lucide React
- **Animations:** Framer Motion

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Content Source

The content is based on the original `stephen.taipei.md` file but adapted to a professional, bright color scheme instead of the original sci-fi theme.
